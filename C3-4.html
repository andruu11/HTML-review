Open Graph

Sos microdatos son un tipo de atributos que se colocan en las etiquetas para que la busqueda de Facebook sea mas eficiente. Cotiene:

  • Prefix: Url de prefix de Open Graph debe ubicarse en la etiqueta HTML.
  • meta : colocar un meta property = "og:type".
  • meta : El meta debe ir en el head.
  • content : dentro la etiqueta meta debe haber un content que referencie el contenido segun el typo de OG.

* El link para la busqueda de prefix y tipos es: ogp.me

* El link para verificar si nuestros microdatos estan aplicados de manera correcta: Sharing Debugger Facebook

Codigo :

					
					<!DOCTYPE html>
					<html prefix="og: http://ogp.me/ns#">
					<head>
						<meta charset="utf-8">
						<meta http-equiv="X-UA-Compatible" content="IE=edge">
						<meta property="og:title" content="Prueba de Open Graph" />
						<meta property="og:type" content="website" />
						<meta property="og:url" content="http://prueba.html" />
						<meta property="og:image" content="http://ia.media-imdb.com/images/rock.jpg" />
						<title>Prueba de open Graph</title>
					</head>
					<body>
					</body>
					</html>
