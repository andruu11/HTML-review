Etiquetas de imagen

1) img

Descripción: La etiqueta img nos sirve para invocar una imagen.

Codigo :

					
					<img src="Assets/img/Pacman.png" alt="Imagen de pacman" width="100px" height="100px">

Ejemplo :

2) srcset px

Descripción: esta etiqueta se utiliza segun la variación de pixeles.

Codigo :

					
					<img srcset="Assets/img/1.png 1x,
					Assets/img/2.jpg 2x"
					>

Ejemplo :

2) srcset w

Descripción: esta etiqueta se utiliza segun la variación del tamaño de la imagen.

Codigo :

					
					<img srcset="Assets/img/1.png 500w,
					Assets/img/2.jpg 800w"
					>
