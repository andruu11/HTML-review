Input text, label y placeholder

- Descripci√≥n: Un input siempre debe ir asociado a un label.

- label: Esta etiqueta nos permite mostrar datos, si dentro de esta etiqueta agrupamos una etiqueta input la estariamos asociando.

- label for: Esta etiqueta nos permite mostrar datos, en caso de no desear agrupar el input dentro la etiqueta label, se debe colocar un id al input y en el atributo for del label indicar el mismo valor, para asi esten asociados.

- placeholder: El atributo placeholder muestra un mensaje dentro del input.

- value: El atributo value otorga un valor por defecto al input.

Codigo :

					
					<form action="script.php" method="post" autocomplete="off">
						<label>Nombres: <input type="text" value="Jose Enrrique"></label><br><br>
						<label for="apellidos">Apellidos</label>

						<input type="text" id="apellidos" name="apellidos" placeholder="complete su apellido" ><br><br>
						<button>Registrar</button>
					</form>
