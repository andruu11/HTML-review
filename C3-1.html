Atributos

1) class

Descripción: Se usa para interactuar con JavaScript y CSS

Codigo :

					
					<div class="contenedor-principal"<Lorem ipsum dolor sit amet, consectetur adipisicing elit. Ad cum ex nesciunt aspernatur, veritatis corrupti sapiente aliquam velit doloremque, magni voluptatem, error quia fugit illo quo adipisci nihil ullam vitae?</div<

Ejemplo :

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Ad cum ex nesciunt aspernatur, veritatis corrupti sapiente aliquam velit doloremque, magni voluptatem, error quia fugit illo quo adipisci nihil ullam vitae?

2) id

Descripción: Es un identificador unico para interactuar JavaScript, CSS y HTML. (No pueden haber dos elementos con el mismo id).

Codigo :

					
					<div id="contenedor-slide">Lorem ipsum dolor sit amet, consectetur adipisicing elit. Minus eaque ullam, culpa, delectus labore ipsa at nihil consectetur explicabo nemo excepturi. Inventore facilis totam reprehenderit deserunt, nostrum id vel nulla!</div>

Ejemplo :

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Veritatis nesciunt tempora velit tenetur est fugiat ducimus doloribus impedit, quaerat amet similique eius, reiciendis facere officiis, ad natus hic architecto quidem!

3) title

Descripción: Sirve para mostrar informacion adicional.

Codigo :

					
					<a href="index.html" title="Volver al inicio"></a>

4) data-nombreatributo

Descripción: Sirve para capturar datos desde HTML a JavaScript o CSS.

Codigo :

					
					<p>El precio de los platanos es:</p>
					<p data-price="20">20</p>

Ejemplo :

El precio de los platanos es:

20

5) role

Descripción: Indican al navegador que rol o papel cumple ese elemento.

Codigo :

					
					<nav role="navigation">
				<ul>
					<li><a href="#">Inicio</a></li>
					<li><a href="#">Nosotros</a></li>
					<li><a href="#">Contactos</a></li>
				</ul>
			</nav>

Ejemplo :

  • Inicio
  • Nosotros
  • Contactos
