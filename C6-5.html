textarea y datalist

1) textarea

Descripción: Campo extenso para escribir un mensaje o un texto largo, el atributo maxlength define el maximo de caracteres permitido, cols y rows el tamaño del textarea.

					
						<textarea name="mensaje" id="mensaje" cols="30" rows="3" maxlength="50"></textarea>

2) datalist

Descripción: Es un almacen de datos que en conjunto o asociado con un input y el atributo list se convierte en un buscador de opciones

Codigo :

					
					<datalist id="paises">
						<option value="Bolivia">		
						<option value="España">
						<option value="Francia">
						<option value="Finlandia">		
					</datalist>
