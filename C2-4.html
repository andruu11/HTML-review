Etiquetas de semantica de texto

1) a href

Descripción: Nos redirecciona a otro archivo

Codigo :

					
					<a href="http://www.google.com">Click aqui para ir a Google</a>

Ejemplo :

Click aqui para ir a Google

2) em

Descripción: Nos sirve para resaltar texto

Codigo :

					
					<em>Esto es importante</em>

Ejemplo :

Esto es importante

3) strong

Descripción: Nos sirve para resaltar un texto en negritas

Codigo :

					
					<strong>Texto en negrita</strong>

Ejemplo :

Texto en negrita

4) small

Descripción: Hacer un texto pequeño, texto de menos importancia

Codigo :

					
					<small>Soy letras pequeñas</small>

Ejemplo :

Soy letras pequeñas

5) cite

Descripción: Para citar a un autor

Codigo :

					
					<p>Ingenieria del Software por <cite>Roger Pressman</cite></p>

Ejemplo :

Ingenieria del Software por Roger Pressman

6) q

Descripción: Para citas breves

Codigo :

					
					<q>Si lo puedes soñar lo puedes hacer</q>

Ejemplo :

Si lo puedes soñar lo puedes hacer

7) dfn

Descripción: Para agregar un significado

Codigo :

					
					<p>HTML <dfn>Hyper Text Markup Languaje</dfn></p>

Ejemplo :

HTML Hyper Text Markup Languaje

8) abbr

Descripción: Para agregar abreviaturas

Codigo :

					
					<abbr title="Java Script">JS</abbr>

Ejemplo :

JS

9) time

Descripción: Para indicar alguna hora

Codigo :

					
					<p>Se anunciara el ganador a las <time>18:45 P.M.</time></p>

Ejemplo :

Se anunciara el ganador a las 18:45 P.M.

10) Superindice y Subindice

Descripción: Superindice aparece de forma pequeña en la parte superior, y el subindice en la parte inferior

Codigo :

					
					<p>(a+b)<sup>2</sup> </p>
					<p>H<sub>2</sub>O</p>

Ejemplo :

(a+b)2

H2O

11) u

Descripción: Para subrrayado

Codigo :

					
					<u>Esto esta subrrayado</u>

Ejemplo :

Esto esta subrrayado

11) u

Descripción: Para subrrayado

Codigo :

					
					<u>Esto esta subrrayado</u>

Ejemplo :

Esto esta subrrayado

12) mark

Descripción: Para resaltar una porcion de texto

Codigo :

					
					<p>Este contenido es <mark>subliminal</mark></p>

Ejemplo :

Este contenido es subliminal

12) span

Es el equivalente a un div solo que se aplica a textos y le apliquemos interaccion con css y js

Codigo :

					
					<p>Este <span>contenido</span> es muy interesante</p>

Ejemplo :

Este contenido es muy interesante

Salto de linea (no recomendable)

					<p>Esto es muy interesante</p><br><br>
					<p>Esto es muy interesante</p>

Esto es muy interesante

Salto de linea (no recomendable)

Salto de linea inteligente

Codigo :

					
					<p>aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa<wbr>aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa</p>
