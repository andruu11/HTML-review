Microdatos

Los microdatos son un tipo de atributos que se colocan en las etiquetas para que la busqueda del navegador sea mas eficiente. Cotiene:

  • Scope : Una etiqueta de contenido.
  • Schema : link de la pagina schema.org segun el esquema seleccionado.
  • Propiedades: El atributo relacionado al schema.

* El link para la busqueda de schemas es: Schema.org

* El link para verificar si nuestros microdatos estan aplicados de manera correcta: Herramienta de Google
