Audio y Video

1) Video

Descripción: La etiqueta img nos sirve para invocar una imagen.

- controls: Etiqueta para mostrar controles.

- autoplay: Etiqueta para reproducir automaticamente.

- poster: Etiqueta para mostrar un banner o imagen antes de reproduccir el video.

- muted: Etiqueta para silenciar el video al reproducir, y asi el usuario pueda activar el sonido.

Codigo :

					
						<video src="Assets/img/Video_prueba.mp4" width="500px" controls poster="Assets/img/Pacman.png" muted></video>

2) audio

Descripción: esta etiqueta se utiliza para reproducir archivos en formato de audio.

- controls: Etiqueta para mostrar controles.

- autoplay: Etiqueta para reproducir automaticamente.

- loop: bucle de reproducción

- preload: precarga contenido al ingresar a la pagina

Codigo :

					
					<audio src="Assets/img/Audio_prueba.mp3" controls></audio>
