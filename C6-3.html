Tipos de input

- input text: Es un campo para introducir texto.

- input email: Es un campo para introducir emails.

- input tel: Es un campo para introducir telefono o celular, mejor visibilidad en moviles.

- input date: Es un campo para introducir fecha .

- input time: Es un campo para introducir horas y minutos .

- input num: Es un campo para introducir solo numeros .

- input url: Es un campo para introducir enlaces o url´s .

- input password: Es un campo para introducir contraseñas .

- input url: Es un campo para introducir datos y se produzca su busqueda .

- input color: Es un campo para introducir colores.

- input color: Es un campo para introducir un valor numerico segun una barra.

- input file: Es un campo para introducir los ficheros o archivos.

- input radio: Es un campo para realizar una seleccion.

- input checkbox: Es un campo para realizar seleccion multiple.

- input reset: Es un campo para limpiar el formulario.
