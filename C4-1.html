Rutas

1) Rutas relativas

Descripción: Son relativas ya que dependen del proyecto

Codigo :

					
					<a href="index.html">Inicio</a><br>
			<a href="/">Inicio</a>

Ejemplo :

Inicio
Inicio

2) Rutas absolutas

Descripción: Son rutas que no varian.

Codigo :

					
					<a href="https://www.google.com">Google</a

Ejemplo :

Google

3) Etiqueta base

Descripción: Sirve para definir una raiz en el proyecto.

Codigo :

					
					<!DOCTYPE html>
					<html>
					<head>
						<meta charset="utf-8">
						<meta http-equiv="X-UA-Compatible" content="IE=edge">
						<base href="http://localhost:7879">
						<title></title>
						</head>
					<body>
						<a href="C4-1.html" class="enlace">C4-1 : Rutas absolutas y relativas</a>
						
					</body>
					</html>
