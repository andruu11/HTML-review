Formularios

- Descripci√≥n: Un formulario es un contenedor de elementos que reciben, muestran y agrupan estos elementos.

- fieldset: Esta etiqueta nos permite agrupar elementos del formulario.

- legend: Esta etiqueta nos permite poner un titulo a la agrupacion fieldset.

- label: Esta etiqueta nos permite mostrar datos.

- input: Esta etiqueta nos permite intoducir datos datos.

- input type submit: Esta etiqueta nos permite enviar datos al script que procesa los datos ingresados en el formulario.

- button: Esta etiqueta nos permite enviar datos al script que procesa los datos ingresados en el formulario.

- input type image: Esta etiqueta nos permite enviar datos al script que procesa los datos ingresados en el formulario, solo que usa como boton una imagen.

					<form action="script.php" method="post" autocomplete="off">

				<fieldset>

				<label>Grado: <input></label><br><br>
				<label>Carrera: <input></label>
