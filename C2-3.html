Etiquetas de contenido

1) p

Descripción: Engloba un parrafo o bloque texto

Codigo :

					
					<p>Lorem ipsum dolor sit amet, consectetur adipisicing elit. Repudiandae aspernatur eveniet optio fugit facere dolor architecto, recusandae enim fugiat impedit odit ducimus, consectetur nobis quos velit aliquid necessitatibus voluptatibus voluptas.</p>

Ejemplo :

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Repudiandae aspernatur eveniet optio fugit facere dolor architecto, recusandae enim fugiat impedit odit ducimus, consectetur nobis quos velit aliquid necessitatibus voluptatibus voluptas.

2) hr

Descripción: Se usa para divir dos contenidos con tematica diferente, ademas de aparecer una linea entre estos

Codigo :

					
					<p>Lorem ipsum dolor sit amet, consectetur adipisicing elit. Repudiandae aspernatur eveniet optio fugit facere dolor</p>
					<hr>
					<p>architecto, recusandae enim fugiat impedit odit ducimus, consectetur nobis quos velit aliquid necessitatibus voluptatibus voluptas.</p>

Ejemplo :

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Repudiandae aspernatur eveniet optio fugit facere dolor

architecto, recusandae enim fugiat impedit odit ducimus, consectetur nobis quos velit aliquid necessitatibus voluptatibus voluptas.

3) pre

Descripción: Se usa para preformatear texto, es decir, mantener el orden y forma del parrafo(saltos de linea, tabulaciones)

Codigo :

					
					<pre>architecto, recusandae enim fugiat impedit odit ducimus, consectetur nobis quos velit aliquid necessitatibus 					voluptatibus voluptas.
					Lorem ipsum dolor sit amet, 
					consectetur adipisicing elit. 
					Repudiandae aspernatur eveniet 
					optio fugit facere dolor</pre>

Ejemplo :

architecto, recusandae enim fugiat impedit odit ducimus, consectetur nobis quos velit aliquid necessitatibus 					voluptatibus voluptas.
					Lorem ipsum dolor sit amet, 
					consectetur adipisicing elit. 
					Repudiandae aspernatur eveniet 
					optio fugit facere dolor

4) blockquote

Descripción: Se usan para las citas o frases

Codigo :

					
					<blockquote> "Si lo puedes soñar lo puedes hacer" </blockquote>

Ejemplo :

"Si lo puedes soñar lo puedes hacer"

5) main

Descripción: Representa el contenido principal en el body diferenciandolo y dandole mas relevancia

Codigo :

					
					<main>
						Yo soy el contenido principal del sitio o todo el contenido central del sitio entre el header y el footer
					</main>

Ejemplo :

Yo soy el contenido principal del sitio o todo el contenido central del sitio entre el header y el footer

6) div

Descripción: No tiene ninguna representacion, sirve como contenedor para la interaccion con css y js

Codigo :

					
					<div>Lorem ipsum dolor sit amet, consectetur adipisicing elit. Suscipit dignissimos, harum numquam facere cum facilis earum. Numquam unde delectus nihil, ut, obcaecati reiciendis soluta dolor deleniti saepe ullam possimus repellendus!</div>

Ejemplo :

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Suscipit dignissimos, harum numquam facere cum facilis earum. Numquam unde delectus nihil, ut, obcaecati reiciendis soluta dolor deleniti saepe ullam possimus repellendus!
