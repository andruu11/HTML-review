Twitter Card

Sos microdatos son un tipo de atributos que se colocan en las etiquetas para que la busqueda de Twitter sea mas eficiente. Tipos de tarjeta:

  • Sumary card: Tarjeta sencilla del sitio.
  • Summary card large: Tarjeta larga descriptiva del sitio.
  • App card : Promocionar alguna App.
  • Play card: Tarjeta para multimedia.

Aplicaci√≥n del twitter card

  • Colocar una etiqueta meta
  • Instanciar en name="twitter:tipo" (El tipo se extrae del sitio twitter card)
  • Instanciar en content el contenido segun el tipo

* El link para la busqueda de tipos es: Twitter Card

* El link para verificar si nuestros microdatos estan aplicados de manera correcta: Card Validator

Codigo :

					
					<!DOCTYPE html>
					<html>
					<head>
						<meta charset="utf-8">
						<meta http-equiv="X-UA-Compatible" content="IE=edge">
						<meta name="twitter:url" content="www.html.com" />
						<meta name="twitter:card"  content="sumary_large_image" />
						<meta name="twitter:image" content="http://www.html.com/images/portada.jpg" />
						<meta name="twitter:description" content="Sitio guia de HTML5" />
						<meta name="twitter:tittle" content="HTML5" />
						<meta name="twitter:site" content="@HTML5" />
						<title>Prueba de Twitter Card</title>
					</head>
					<body>
					</body>
					</html>
