Select, option, optgroup

- select: Nos sirve para mostrar una lista de datos y seleccionar una opcion.

- select multiple: Nos sirve para mostrar una lista de datos y seleccionar multiples opciones.

- optgroup label="nombredelgrupo": este etiqueta y atributo nos sirven para agrupar elementos de un select .

Codigo :

					
					<form action="script.php" method="post" autocomplete="off">
						<select name="paises" id="paises">
							<option value="Bolivia">Bolivia</option>
							<option value="Argentina">Argentina</option>
							<option value="Chile">Chile</option>
							<option value="Peru">Peru</option>
						</select>

						<select multiple name="categorias" id="categorias">
							<option value="Correr">Correr</option>
							<option value="Cantar">Cantar</option>
							<option value="Bailar">Bailar</option>
							<option value="Trotar">Trotar</option>
						</select>

						<select name="paises" id="paises">
							<optgroup label="America">
							<option value="Bolivia">Bolivia</option>
							<option value="Argentina">Argentina</option>
							<option value="Chile">Chile</option>
							<option value="Peru">Peru</option>
							</optgroup>
							<optgroup label="Europa">
							<option value="Francia">Francia</option>
							</optgroup>
						</select>
					</form>
