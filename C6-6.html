Atributos de Formularios

- required: atributo para solicitar llenado obligatorio de un input.

- disabled: atributo para desactivar un input.

- checked: atributo para autoseleccionado en un input checkbox o un input radio.

- selected: atributo para seleccionar un campo del select o combobox.

- multiple: atributo para hacer un select de opcion multiple.

- autofocus: atributo para al iniciar una pagina seleccione automaticamente ese campo.

- readonly: atributo de solo lectura.

- maxlength: atributo para determinar el numero maximo de caracteres a introducir en un input.

- minlength: atributo para determinar el numero minimo de caracteres a introducir en un input.

- min: atributo para determinar el numero minimo a introducir en un input tipo number.

- max: atributo para determinar el numero masximo a introducir en un input tipo number.

- step: atributo para determinar el intervalo a introducir en un input tipo number.

- tabindex: atributo para determinar el orden de tabulacion entre input.

- autocomplete off: atributo que se ubica en la etiqueta form, para desactivar el autocompletado en los input.
